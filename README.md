# FrontEnd Challenge - 

## Introduction

To start this project, you can:

-Clone this repo.

- Inside the folder run `npm install` to get all the modules from the `package.json` file.

```bash
$ npm install
```

Run `npm start`  

```bash
$ npm start
```
